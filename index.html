Machines Learn. We Interpret Them.


What does interpreting an application mean? Interpreting an application based on a machine learning model means explaining the recommendations or actions of this model using a language that can be understood by its users.

Why do I need reliable machines? Because a machine learning model can be so complex that we humans may not have the ability to interpret it directly. So we need machines that we can trust on to help us with this task.

How it works? We use proprietary techniques to develop an interpretability system that will increase the ability of users to interpret the predictions and actions of your application.

Who are you? We are a group of AI specialists specializing in interpretability of machine learning models. We are organized into a structure with five research laboratories specializing in human-centered computing, machine learning, optimization, programming language and software engineering.
